# 前言
在整理部落格資料的時候發現封面的圖片需求量是有的，
抓網上的圖解析度常會各種混亂甚至浮水印或版權等問題，
所以最一開始的想法是用SD模型來生圖一來可以完美解決解析度問題，二來可以根據XP自行調整還是比較香的，
但是實際上穩定擴散卻不是那麼穩定inpainting功能或者img2img這類的常會生出一些過於前衛的藝術，
像是兩顆頭的美少女或者三腳獸這類超越人類審美的藝術。

後來在使用了幾次後發現SD雖然能給整體佈局很美的view但細部還是得靠工人智慧去細尻，
細尻的部分基於個人習慣我還是比較常用PS來做，設計工具還是比較全的。
一開始有考慮用Adobe Firefly去做，但風格上個人還是偏愛二刺螈，很快就又回到了SDwebui上了。

這篇文章主要是面向繪圖愛好者
主要以**安裝**跟**使用**的觀點去紀錄

# 前置
### 硬體(電腦)
任何AI的東西都是很吃計算的，顯卡越好意味著算圖更快。
目前主流用來跑的顯卡不外乎就是NVIDIA，其他能用但我都不推也不適合非開發人員。

在硬體方面會建議用桌上型電腦來做這事情，原因同上不外乎就是散熱跟計算效能，筆電不推是因為會讓電池短壽。
以下是建議配置:
1. 在RAM(記憶體)的方面會建議盡量堆高實際上PS圖層一多的時候16GB就可能吃超過一半了在加上網頁跟SD的運作會接近滿載，因此在記憶體方面我會建議32GB以上。
2. GPU(顯卡)則是需要考慮到VRAM的大小，VRAM會決定模型一次能計算多大的圖這部分在[NVIDIA網頁](https://www.nvidia.com/zh-tw/geforce/graphics-cards/40-series/)能看，一般消費級的顯卡以RTX3090跟RTX4090的VRAM為最大的(24GB)，在這類算圖的模型我會建議至少買VRAM 12GB以上的GPU

### 軟體
作業系統就windows吧 10或11 

在軟體方面有兩個需要進行安裝跟下載
1. Python (版本>3.10)  
   https://www.python.org/downloads/  
   3.11.5 x64 [下載](https://www.python.org/ftp/python/3.11.5/python-3.11.5-amd64.exe)  
   3.10.0 x64 (跟本篇一樣的) [下載](https://www.python.org/ftp/python/3.10.0/python-3.10.0-amd64.exe)

2. Git  
   https://git-scm.com/download/win

3. Photoshop  
   需要安裝24.0以後的版本 (PS2023以後的版本)

其他一些非必要的軟體:  
NVIDIA CUDA 跟 cudnn開發包 (開發者或有需要做finetune)
